---

Un **plan de producción** es un escenario de planificación. Copia operaciones abiertas, recursos, restricciones, turnos, fechas y contexto de stock para calcular una propuesta de secuencia.

El plan no cambia la ejecución real hasta que lo aplicas.

## Para qué sirven

| Uso | Resultado |
| --- | --- |
| Evaluar capacidad | Ver si el trabajo cabe en recursos y turnos disponibles. |
| Comparar alternativas | Probar fechas, prioridades o alcances antes de decidir. |
| Detectar bloqueos | Encontrar operaciones sin recurso, material o secuencia viable. |
| Aplicar una propuesta | Actualizar operaciones reales cuando el plan es ejecutable. |

## Ciclo del plan

| Acción | Qué hace |
| --- | --- |
| Crear | Define alcance, fechas, órdenes o recursos incluidos. |
| Actualizar | Carga la realidad operativa actual. |
| Resolver | Calcula una propuesta de secuencia. |
| Revisar | Muestra retrasos, bloqueos o datos faltantes. |
| Aplicar | Traslada la propuesta a operaciones reales abiertas. |

Si cambian órdenes, recursos, turnos o materiales después de resolver, actualiza o resuelve el plan de nuevo antes de aplicarlo.

## Estados del cálculo

| Estado | Significado |
| --- | --- |
| **Pendiente** | El plan está creado o actualizado, pero todavía no se está resolviendo. |
| **Procesando** | Bold está calculando una propuesta de secuencia. |
| **Solución encontrada** | Existe una propuesta revisable y aplicable. |
| **Fallido** | El cálculo no terminó correctamente. |
| **Sin solución factible** | No se encontró una secuencia válida con los datos y restricciones actuales. |

Consulta [Estados de fabricación](/es/conceptos/fabricacion/estados-de-fabricacion) para distinguir estados de planes y estados de ejecución real.

## Qué no modifica

Un plan no debe cambiar operaciones finalizadas ni canceladas. Tampoco sustituye los registros reales de ejecución. Sirve para planificar trabajo abierto y trasladar una propuesta aprobada.

## Relación con planificación

Planificación responde qué falta y cuándo falta. El plan de producción responde cómo encajar el trabajo de fabricación en recursos y turnos disponibles.
