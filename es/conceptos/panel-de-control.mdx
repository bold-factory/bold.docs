## Áreas administrativas

| Área | Qué gestiona |
| --- | --- |
| Cuenta y plan | Datos de suscripción, límites y uso visible para administradores. |
| Empresa | Datos generales y configuración de negocio. |
| Acceso | Usuarios, perfiles, permisos y claves de integración. |
| Eventos | avisos externos, notificaciones, automatizaciones y auditoría. |
| Datos | Importaciones, archivos y mantenimiento de información. |
| Diseños | Plantillas, etiquetas y formatos de salida. |

## Reglas generales

Bold valida permisos antes de ejecutar acciones protegidas. Si falta un permiso, la interfaz puede ocultar o bloquear acciones.

Configura acceso y claves de integración con el menor alcance suficiente. Cambios administrativos pueden afectar stock, órdenes, costes, integraciones y trazabilidad.
