Los **informes** de Smart Factory usan Power BI embebido para analizar datos de Bold con visualizaciones, filtros y campos preparados.

Úsalos para análisis periódico, seguimiento de indicadores y vistas de dirección.

## Qué representan

| Concepto | Uso |
| --- | --- |
| Informe | Vista analítica con páginas, visualizaciones y filtros. |
| Modelo de datos | Conjunto de entidades preparado para analizar datos de Bold. |
| Informe base | Punto de partida para crear o duplicar informes. |
| Informe editable | Copia que puedes adaptar si tienes permisos. |

## Cuándo usarlos

| Necesidad | Usa |
| --- | --- |
| Analizar tendencias, comparar periodos o preparar reportes | **Informes**. |
| Ver estado vivo de planta, operaciones en curso u OEE operativo | [Cuadros de mando de fabricación](/es/ayuda/produccion/cuadros-de-mando). |
| Investigar un registro concreto | El módulo operativo correspondiente, como órdenes, stock o partes. |

Los informes no son una vista en tiempo real. Se actualizan según la programación del entorno.

## Permisos

| Acción | Permiso |
| --- | --- |
| Abrir y listar informes | `Analytics.Reports.Read` |
| Crear, duplicar o eliminar informes | `Analytics.Reports.Write` |

## Reglas de uso

- Duplica informes antes de hacer cambios relevantes.
- Usa nombres orientados al uso, como `Coste por activo` o `Producción semanal`.
- Revisa fechas, estados, unidades y filtros antes de tomar decisiones.
- Usa cuadros de mando cuando necesites estado operativo en vivo.
