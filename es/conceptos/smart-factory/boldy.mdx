**Boldy** es el asistente conversacional de Bold. Permite consultar información en lenguaje natural, resumir datos y orientarte sin navegar por todos los menús.

## Cuándo usar Boldy

| Necesidad | Ejemplo |
| --- | --- |
| Localizar datos | `¿Qué pedidos de venta están pendientes esta semana?` |
| Resumir información | `Resume las órdenes de fabricación con retraso.` |
| Entender un proceso | `Explícame qué significa stock proyectado.` |
| Preparar una búsqueda | `Ayúdame a encontrar partes de mantenimiento de la envasadora.` |

Boldy responde con el contexto y permisos disponibles para tu usuario. Si faltan datos, debe pedir aclaración o indicar qué falta.

## Conversaciones

Una conversación conserva contexto. Úsala para profundizar en el mismo objetivo. Crea una conversación nueva cuando cambie el tema o no quieras arrastrar decisiones anteriores.

| Buen patrón | Por qué ayuda |
| --- | --- |
| Indicar módulo | Evita respuestas demasiado generales. |
| Indicar fechas o estado | Acota el análisis. |
| Incluir códigos, artículos, activos o ubicaciones | Reduce ambigüedad. |
| Verificar registros citados | Protege decisiones operativas. |

## Diferencia con agentes

| Herramienta | Uso principal |
| --- | --- |
| Boldy | Conversación directa y rápida con el asistente personal. |
| Tareas con agentes | Trabajos delegados a agentes especializados, con estado, archivos, herramientas, aclaraciones y créditos. |

Usa Boldy para explorar o preguntar. Usa una tarea con agente cuando el trabajo sea repetible, largo, necesite archivos o deba quedar auditado como ejecución delegada.
