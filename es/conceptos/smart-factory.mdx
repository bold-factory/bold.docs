## Cuándo usar cada herramienta

| Necesidad | Usa |
| --- | --- |
| Analizar tendencias o preparar reportes periódicos | **Informes**. |
| Preguntar, explorar o resumir datos en conversación | **Boldy**. |
| Delegar un trabajo largo, repetible o auditable | **Agentes**. |
| Ejecutar agentes sin crear tareas manualmente | **Automatizaciones**. |

Los informes sirven para análisis periódico. No sustituyen a vistas operativas en vivo, como cuadros de mando de fabricación o pantallas de ejecución.

Boldy sirve para interacción directa. Las tareas con agentes sirven para trabajos más largos, trazables o automatizados.

## Automatizaciones

Una **automatización** ejecuta un agente sin que tengas que crear la tarea manualmente. Cada ejecución crea una **tarea con agente** con historial, estado, resultado y créditos propios.

| Tipo | Cuándo se ejecuta | Qué conserva |
| --- | --- | --- |
| Periódica | En una frecuencia diaria, semanal, mensual, trimestral, semestral o anual. | Próxima ejecución, última ejecución y número de ejecuciones. |
| Por evento | Cuando Bold publica un evento compatible y se cumplen sus condiciones. | Evento, versión, condición, responsable y tarea generada. |

Usa automatizaciones para tareas repetibles, revisiones programadas, alertas operativas o comprobaciones que deben quedar auditadas. Revisa las primeras ejecuciones antes de depender de ellas en procesos críticos.

Los agentes de sistema no se automatizan. Usa agentes personalizados con instrucciones, permisos y nivel de inteligencia adecuados para el trabajo.

## Estados y reglas de tareas

| Estado | Qué significa |
| --- | --- |
| **Pendiente** | La tarea está creada y espera procesamiento. |
| **En ejecución** | El agente está trabajando. |
| **Requiere acción** | El agente necesita una aclaración o decisión. |
| **Completada** | El resultado está disponible para revisar. |
| **Fallida** | La tarea terminó con error o no pudo completarse. |

Las tareas con agentes conservan instrucciones, mensajes, archivos adjuntos, herramientas usadas, aclaraciones, resultado y créditos. Caducan tras 28 días de inactividad.

## Relación con Panel de control

Smart Factory cubre informes, Boldy, agentes, tareas y automatizaciones. Panel de control queda para cuenta, empresa, usuarios, permisos, claves API, webhooks, notificaciones, datos y diseños.
