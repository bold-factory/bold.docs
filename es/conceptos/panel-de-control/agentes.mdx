Desde Panel de control configuras los agentes que después ejecutan tareas en Smart Factory.

## Qué se configura

| Campo | Uso |
| --- | --- |
| Nombre | Identifica el agente para usuarios y tareas. |
| Nivel de inteligencia | Ajusta capacidad de razonamiento y consumo de créditos. |
| Instrucciones | Define objetivo, límites y forma de responder. |
| Permitir preguntas | Permite que el agente pida aclaraciones cuando falte información. |
| Memoria | Guarda contexto persistente del agente cuando está habilitada. |
| Permisos | Determinan qué herramientas y datos puede usar. |
| Automatizaciones | Ejecutan el agente de forma programada. |

## Nivel de inteligencia

| Nivel | Cuándo usarlo |
| --- | --- |
| Bajo | Consultas directas, resúmenes simples o tareas muy acotadas. |
| Medio | Tareas operativas habituales con equilibrio entre coste y razonamiento. |
| Alto | Análisis con varios pasos, instrucciones ambiguas o tareas que combinan varias áreas. |

Empieza por el nivel más bajo que cumpla la tarea. Sube el nivel si el agente se queda corto, omite pasos o necesita razonar con más contexto.

## Automatizaciones

Las automatizaciones ejecutan agentes de forma recurrente. Cada ejecución crea una tarea con agente y conserva historial propio.

| Elemento | Uso |
| --- | --- |
| Frecuencia | Diaria, semanal, mensual, trimestral, semestral o anual. |
| Responsable | Usuario asociado al seguimiento de la automatización. |
| Próxima ejecución | Fecha calculada para el siguiente lanzamiento. |
| Estado | Indica si la automatización está habilitada y puede ejecutarse. |

Revisa las primeras tareas generadas antes de confiar en la automatización.

<Warning>
  Un agente solo debe tener los permisos que necesita. Si le das permisos amplios, podrá acceder a más datos y acciones durante sus tareas.
</Warning>
