Los **diseños** definen formatos de salida para documentos, etiquetas o identificadores que usa el equipo en planta, almacén o administración.

## Conceptos

| Concepto | Uso |
| --- | --- |
| Diseño | Plantilla reutilizable para generar una salida. |
| Etiqueta | Formato imprimible para identificar artículos, lotes, ubicaciones, estaciones u otros objetos. |
| Valor dinámico | Dato de Bold insertado en el diseño. |
| Código QR o de barras | Representación escaneable para acelerar identificación. |

## Reglas de diseño

Una plantilla debe ser legible en el tamaño real de impresión y contener solo la información necesaria para identificar el objeto.

Antes de usar una plantilla de forma masiva, descarga una muestra y pruébala con el lector que usará el equipo.
