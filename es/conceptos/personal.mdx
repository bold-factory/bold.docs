## Entidades

| Concepto | Uso |
| --- | --- |
| Empleado | Persona operativa que puede registrar tiempos, recibir asignaciones o participar en costes. |
| Jornada laboral | Plantilla de tramos de trabajo. |
| Calendario local | Excepciones por festivos, cierres o días especiales. |
| Turno de personal | Programación que asigna jornadas y calendarios a empleados. |
| Turno de trabajo | Turno usado en operación de planta o fabricación. |
| Periodo de trabajo | Tramo real de actividad o presencia registrado para un empleado. |
| Tarea | Trabajo operativo asignado o calculado para una persona. |

## Usuario y empleado

Un usuario controla acceso. Un empleado representa trabajo operativo. Pueden estar relacionados, pero no son el mismo concepto.

Usa un usuario para entrar en Bold, recibir permisos y administrar módulos. Usa un empleado para registrar mano de obra, calcular costes, asignar operaciones o medir presencia. Consulta [Usuarios, roles y permisos](/es/conceptos/panel-de-control/usuarios-roles-y-permisos) para la parte de acceso.

## Jornadas, calendarios y turnos

Una jornada define el patrón normal de trabajo. Un calendario local define excepciones. Un turno aplica esas reglas a personas o equipos.

| Concepto | Qué decide |
| --- | --- |
| Jornada laboral | Qué tramos componen un día tipo. |
| Calendario local | Qué excepciones alteran el patrón normal. |
| Turno | Qué ciclo repetitivo se aplica a empleados o equipos. |
| Relación operativa | Cómo esa capacidad afecta a planificación, ejecución y costes. |

## Impacto en otros módulos

| Módulo | Impacto |
| --- | --- |
| Producción | Asignación de empleados, registros de tiempo y análisis de eficiencia. |
| Mantenimiento | Partes de trabajo, tiempo de intervención y empleado principal. |
| Costes | Coste horario de empleados y coste real de mano de obra. |
| Panel de control | Relación entre empleados, usuarios, roles, permisos e invitaciones. |
| Modo de trabajo | Identificación del empleado activo y tareas disponibles en planta. |
