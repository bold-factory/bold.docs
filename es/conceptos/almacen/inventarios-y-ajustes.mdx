Los **inventarios** y **ajustes** regularizan diferencias entre stock registrado y realidad física. Deben explicar una diferencia, no sustituir operaciones que sí tienen origen.

## Inventario

Un inventario compara stock esperado con conteo físico. Sirve para cierres, auditorías o revisiones por zona.

| Dato | Significado |
| --- | --- |
| Stock previo | Cantidad registrada antes del conteo. |
| Cantidad contada | Lectura física. |
| Ajuste previsto | Diferencia que se aplicará al finalizar. |
| Alcance | Ubicaciones, artículos o lotes incluidos. |

Usa inventarios cuando hay varias líneas, varias ubicaciones o un proceso formal de conteo.

## Ajuste de stock

Un ajuste corrige una diferencia puntual con causa conocida. Normalmente afecta a una combinación concreta de artículo, ubicación y lote.

Úsalo para correcciones pequeñas o incidencias claras. Si la diferencia afecta a una zona completa, usa inventario.

## Movimiento interno

Un movimiento interno cambia dónde está el material, pero no debería cambiar la cantidad total del artículo.

| Caso | Concepto correcto |
| --- | --- |
| El material existe, pero está en otra ubicación | Movimiento interno. |
| El material se contó y falta cantidad | Inventario o ajuste. |
| Falta registrar una recepción | Recepción. |
| Falta registrar un consumo | Consumo o ejecución correspondiente. |

## Riesgos

Un ajuste manual puede ocultar errores de recepción, consumo, envío o ubicación. Antes de ajustar, revisa movimientos recientes, documentos pendientes y lotes.
