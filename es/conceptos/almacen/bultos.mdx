Un **bulto** representa una unidad física movible dentro del almacén, como un palet, caja, carro, saco, contenedor o kit.

El bulto no sustituye al stock. Agrupa líneas de stock para que el equipo pueda identificar y mover una unidad física completa.

## Qué contiene

| Dato | Uso |
| --- | --- |
| Código | Identifica el bulto mediante etiqueta, QR o búsqueda. |
| Tipo de contenedor | Indica formato físico: caja, palet, carro o similar. |
| Ubicación actual | Dónde está el bulto. |
| Contenido | Artículos, lotes y cantidades incluidos. |
| Estado | Activo, vacío, cerrado o no operativo según el uso. |

Un bulto puede contener varios artículos y lotes si físicamente viajan juntos y el equipo puede identificarlos sin ambigüedad.

## Cuándo usar bultos

Usa bultos cuando:

- mueves material agrupado;
- necesitas imprimir etiquetas por unidad física;
- preparas kits, palets o cajas para producción o envío;
- haces inventario escaneando unidades físicas;
- quieres conocer la ubicación de un contenedor completo.

No uses bultos para cada unidad si el equipo opera por cantidad suelta. Añadir bultos sin necesidad aumenta registros sin mejorar trazabilidad.

## Relación con ubicaciones

Un bulto tiene una ubicación actual. Cuando el bulto se mueve, su contenido se considera asociado a la nueva ubicación.

Si sacas parte del contenido, el bulto conserva lo que queda. Si queda vacío, puede dejar de estar activo.

## Relación con tipos de contenedores

El tipo de contenedor clasifica el formato del bulto. Sirve para distinguir palets, cajas, carros, jaulas o formatos retornables.

El bulto es la unidad concreta. El tipo de contenedor es la plantilla o categoría.
