La **valoración de inventario** calcula el valor económico del stock registrado.

Bold valora líneas de stock usando cantidad disponible y coste asociado, normalmente por lote u origen.

## Qué se valora

| Dato | Uso |
| --- | --- |
| Artículo | Referencia valorada. |
| Ubicación | Lugar donde está el stock. |
| Lote | Origen y coste del conjunto trazable. |
| Cantidad | stock positivo registrado. |
| Coste unitario | Coste aplicado a esa línea. |
| Valor | Cantidad por coste unitario. |

## Fórmula

```text
Valor de línea = stock disponible x coste unitario
Valor total = suma de líneas valoradas
```

## Diferencias habituales

Dos líneas del mismo artículo pueden tener costes distintos si pertenecen a lotes distintos. Esto permite conservar diferencias por compra, fabricación o regularización.

Si un lote no tiene coste, revisa su origen. Puede venir de una recepción sin precio, un ajuste, un consumo temporal o una fabricación incompleta.
