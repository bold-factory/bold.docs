## Flujo conceptual

| Etapa | Qué decide |
| --- | --- |
| Receta | Cómo se fabrica un producto y qué versión está aprobada. |
| Petición de fabricación | Qué artículo necesitas, qué cantidad y para cuándo. |
| Orden de fabricación | Qué trabajo real se ejecutará para cubrir la petición. |
| Operaciones | Qué pasos productivos componen la orden. |
| Recursos y turnos | Dónde y cuándo puede ejecutarse cada operación. |
| Secuenciación y planes | Qué propuesta de ejecución conviene aplicar. |
| Ejecución | Qué se hizo, quién participó, qué se consumió y qué se produjo. |
| Cuadros de mando | Qué rendimiento, desviaciones o bloqueos muestra la planta. |

## Relaciones con otros módulos

| Módulo | Relación |
| --- | --- |
| Catálogo | Las recetas se definen sobre productos y fabrican artículos. |
| Planificación | Las recomendaciones pueden crear peticiones de fabricación. |
| Almacén | Los consumos reducen stock y la producción aumenta stock. |
| Personal | Los empleados registran trabajo, presencia o mano de obra. |
| Costes | Los tiempos, consumos y cantidades reales alimentan el coste. |

## Reglas principales

- Una receta define la estructura aprobada, pero una orden guarda la ejecución real.
- Una petición expresa necesidad; una orden expresa trabajo ejecutable.
- Una operación necesita planificación suficiente antes de ejecutarse.
- Las restricciones entre operaciones no deben duplicarse ni formar ciclos.
- Los recursos, turnos y materiales condicionan la secuenciación.
- Los controles obligatorios pueden bloquear inicio o cierre de operaciones.
- Los consumos, producciones, pausas y tiempos protegen la trazabilidad.
