Un **empleado** representa una persona operativa. Sirve para registrar tiempos, asignar trabajo, calcular mano de obra y analizar productividad.

No todo empleado necesita usuario. No todo usuario necesita empleado.

## Para qué se usa

| Área | Uso del empleado |
| --- | --- |
| Producción | Asignación, ejecución y registros de tiempo. |
| Mantenimiento | Tiempo de intervención y empleado principal. |
| Costes | Coste horario de mano de obra. |
| Planificación | Capacidad cuando se usa disponibilidad de personas. |
| Informes | Productividad, eficiencia y actividad por persona. |

## Usuario frente a empleado

| Usuario | Empleado |
| --- | --- |
| Accede a Bold. | Representa trabajo operativo. |
| Tiene roles y permisos. | Tiene calendario, turno y coste horario. |
| Puede administrar datos. | Puede imputar tiempo o recibir asignaciones. |

Vincula usuario y empleado cuando una persona entra en Bold y además registra trabajo propio.

## Coste horario

El coste horario debe representar el coste empresarial que quieres imputar. Puede incluir salario bruto, cotizaciones, costes indirectos o el criterio interno que uses para fabricación y mantenimiento.

Si falta coste horario, los informes de mano de obra real pueden quedar incompletos.
