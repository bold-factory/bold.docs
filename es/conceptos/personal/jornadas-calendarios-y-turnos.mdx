Las jornadas, calendarios y turnos explican cuándo trabaja una persona o equipo.

Sirven para capacidad, asignaciones, tiempos esperados y análisis de productividad.

Para comparar planificación con actividad real, consulta [Periodos, tareas y presencia](/es/conceptos/personal/periodos-tareas-y-presencia).

## Conceptos

| Concepto | Uso |
| --- | --- |
| Jornada laboral | Plantilla de tramos de trabajo de un día tipo. |
| Calendario local | Excepciones por festivos, cierres o días especiales. |
| Turno | Ciclo repetitivo que aplica jornadas durante varios días. |
| Asignación | Relación de empleado o equipo con su patrón de trabajo. |

## Jornada laboral

Una jornada define períodos de trabajo. Puede representar mañana, tarde, noche, descanso o turno partido.

Si un tramo termina al día siguiente, úsalo para turnos nocturnos.

Los períodos de una misma jornada no deben solaparse. Si la duración parece incorrecta, revisa si algún tramo cruza medianoche.

## Calendario local

El calendario local introduce excepciones: festivos, cierres, jornadas especiales o cambios puntuales.

Las excepciones deben ser claras y tener prioridad sobre el patrón repetitivo.

Si una planta tiene festivos distintos, usa un calendario separado. Si una fecha muestra otra jornada, revisa si existe una asignación manual con más prioridad.

## Turnos

Un turno define un ciclo. Por ejemplo: cinco días de mañana, dos de descanso, cinco de tarde.

Cada segmento indica qué jornada aplica y cuántos días consecutivos dura.

Cada segmento debe durar al menos un día. La fecha de inicio del turno define cómo se alinea el ciclo con el calendario.

## Prioridad de resolución

| Prioridad | Regla |
| --- | --- |
| 1 | Asignación manual o excepción directa. |
| 2 | Calendario local. |
| 3 | Patrón repetitivo del turno. |
| 4 | Jornada por defecto, si existe. |
