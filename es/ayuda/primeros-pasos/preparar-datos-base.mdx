---
title: "Preparar datos base"
description: "Crea el catálogo, los empleados y los recursos mínimos para lanzar una primera fabricación."
---

Prepara solo los datos que necesitas para completar el primer recorrido. El objetivo es que un pedido pueda convertirse en una OF ejecutable, no cargar toda la operativa de la empresa.

Usa un producto habitual y fácil de reconocer. Si empiezas con un caso raro, acabarás resolviendo excepciones antes de validar el flujo principal.

## Resultado esperado

Al terminar, deberías tener:

- un producto fabricable;
- un artículo terminado asociado al producto;
- los componentes que quieras consumir en la receta;
- al menos un empleado operativo;
- un proceso y un recurso de producción compatible;
- una ubicación básica si vas a registrar consumos o producción con stock.

## Crear la base mínima

<Steps>
  <Step title="Crea el producto">
    Entra en **Catálogo > Productos** y crea la familia del artículo terminado. Escoge un producto común, con código estable y nombre claro.
  </Step>
  <Step title="Crea el artículo terminado">
    Entra en **Catálogo > Artículos** y crea la referencia fabricable. Configura el origen de aprovisionamiento como **Fabricación**.
  </Step>
  <Step title="Crea los componentes">
    Añade los artículos que consumirá la receta. Para el primer recorrido, usa pocos componentes y cantidades simples.
  </Step>
  <Step title="Crea un empleado operativo">
    Entra en **Personal > Empleados** y crea la persona que registrará trabajo en planta. Vincula un usuario si esa persona debe iniciar sesión.
  </Step>
  <Step title="Crea proceso y recurso">
    Crea un proceso, como **Montaje**, **Corte** o **Envasado**. Después crea una estación, máquina, línea o puesto que pueda ejecutar ese proceso.
  </Step>
  <Step title="Crea una ubicación básica">
    Añade una ubicación inicial si vas a consumir componentes o registrar stock producido. Puedes separar zonas de recepción, producción y expedición más adelante.
  </Step>
</Steps>

## Decisiones recomendadas

| Decisión | Recomendación para el primer recorrido |
| --- | --- |
| Producto | Usa una familia simple con una variante. |
| Artículo terminado | Márcalo como fabricable y usa una unidad conocida. |
| Componentes | Crea solo los materiales que quieras ver consumidos. |
| Lotes | Déjalos fuera salvo que la trazabilidad sea imprescindible. |
| Recurso | Usa una estación o puesto con nombre reconocible para planta. |
| Empleado | Crea una persona operativa real o de prueba que el equipo identifique. |

<Tip>
  Si no estás seguro de cómo modelar una variante, crea primero una referencia simple. Corrige el patrón antes de cargar muchas familias parecidas.
</Tip>
