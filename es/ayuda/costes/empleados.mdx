Usa **Costes de empleados** para mantener el coste por hora que Bold aplica a registros de mano de obra. Este coste afecta a órdenes de fabricación, operaciones y partes de mantenimiento.

## Qué estás revisando

| Campo | Qué significa |
| --- | --- |
| Código y nombre | Identifican al empleado. |
| Coste/hora | Coste total para la empresa por hora efectiva de trabajo. |

El coste/hora debe representar el coste empresarial, no solo el salario neto. Incluye salario bruto, cotizaciones y otros costes laborales que quieras imputar a fabricación o mantenimiento.

## Calcular un coste horario

<Steps>
  <Step title="Calcula el coste anual">
    Suma salario bruto anual y costes empresariales asociados.
  </Step>
  <Step title="Define horas efectivas">
    Usa las horas anuales de trabajo que aplica tu empresa. Para una jornada de 40 horas semanales, una referencia aproximada son 1.826 horas anuales.
  </Step>
  <Step title="Divide coste entre horas">
    Coste/hora = coste anual / horas anuales efectivas.
  </Step>
  <Step title="Registra el valor">
    Abre el empleado desde **Costes** > **Empleados** o desde **Personal** y guarda el coste/hora.
  </Step>
</Steps>

## Ejemplo

Un empleado tiene un coste anual estimado de `42.000 €`. Si usas `1.826` horas anuales efectivas, el coste horario es:

```text
42.000 / 1.826 = 23,00 €/h
```

Registra `23,00 €/h` como coste del empleado. Cuando ese empleado impute `2 h` a una operación, Bold puede calcular `46,00 €` de mano de obra real.

## Cómo lo usa Bold

| Flujo | Uso del coste de empleado |
| --- | --- |
| Fabricación | Calcula coste real de mano de obra cuando se registran tiempos por empleado. |
| Mantenimiento | Calcula coste real de partes con horas de trabajo registradas. |
| Coste previsto | Puede usar promedios cuando todavía no se sabe qué empleado ejecutará una operación. |

<Tip>
  Revisa costes horarios cuando cambien convenios, salarios o calendarios. Un coste desactualizado distorsiona todas las comparativas de mano de obra.
</Tip>

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| Mano de obra real a cero | El empleado no tiene coste/hora o el tiempo no se registró sobre un empleado. |
| Costes muy altos o bajos | Revisa si dividiste por horas anuales efectivas, no por horas semanales o mensuales. |
| El coste previsto no coincide con el real | El previsto puede usar promedios; el real usa los empleados y tiempos registrados. |
| No puedes editar el coste | Revisa permisos de costes de empleados. |
