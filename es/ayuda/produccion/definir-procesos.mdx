---
title: "Definir procesos"
description: "Crea procesos y asigna estaciones principales o auxiliares para ejecutar cada tipo de trabajo."
---

Los **procesos** son categorías reutilizables de trabajo productivo, como corte, montaje, soldadura, pintura o inspección.

Sirven para conectar una operación de receta con los recursos capaces de realizarla. Una operación indica qué proceso necesita; Bold puede proponer o limitar las estaciones de trabajo compatibles con ese proceso.

## Pasos a seguir

**Ruta:** **Fabricación \> Procesos \> Añadir nuevo tipo**

**Nombre:** usar un nombre corto, claro y técnico que identifique de inmediato la acción global de transformación o el producto final que se obtiene.

<Note>
  **Ejemplos:** prensado, embalaje, perforación o control de calidad.
</Note>

**Descripción:** sirve para acotar técnicamente el alcance del proceso. Mientras que el nombre es genérico, la descripción detalla las tareas reales que ocurren dentro de él.

<Note>
  **Ejemplo:** para el proceso `Unión de piezas` del sector textil, usa la descripción `Confección y costura (remallado y pespunte plano) de los paneles frontales y traseros`.
</Note>

## Asignar estaciones de trabajo al proceso

Una vez creado el proceso, debes indicar **en qué máquinas o lugares físicos** de la fábrica se puede realizar. Puedes asignar una sola estación o varias si el proceso se puede hacer en distintos sitios.

**Ruta:** **Fabricación \> Procesos \> Proceso creado**

**+ Asignar estación**

**Estación de trabajo:** selecciona en el desplegable el nombre de la máquina, línea o mesa de trabajo donde se ejecuta este proceso.

<Note>
  **Ejemplo:** si el proceso es `Soldadura`, selecciona **Puesto de soldadura TIG 02**.
</Note>

<Tip>
  Si no aparece la estación que buscas, asegúrate de haberla creado previamente en el apartado **Crear recursos**.
</Tip>

**Eficiencia (%):** indica el ritmo de trabajo de esta máquina en concreto para este proceso, tomando el 100% como la velocidad estándar o ideal.

- **Si la máquina trabaja al ritmo normal:** déjalo en `100 %`.
- **Si es una máquina antigua o más lenta:** baja el porcentaje (por ejemplo, `80 %`). Así el sistema calculará tiempos de producción más largos y realistas cuando se use esta máquina.
- Si no necesitas este nivel de detalle, deja el valor por defecto: `100 %`.

**Tipo de estación (Principal o Auxiliar):** define el rol de esta máquina para el proceso.

- **Principal:** es la estación habitual donde se saca el trabajo en el día a día. Puedes marcar varias como principales si tienes, por ejemplo, tres líneas idénticas trabajando a la vez.
- **Auxiliar:** es una estación de reserva, un cuello de botella secundario o un puesto manual que solo usas si la máquina principal está rota, saturada o para pedidos muy pequeños.

<Note>
  **Ejemplo:** para un proceso de `Embotellado`, marca la `Línea automática 01` como **Principal** y la `Mesa de envasado manual` como **Auxiliar** para emergencias o lotes pequeños.
</Note>
