Esta página resume los permisos funcionales que puedes asignar mediante roles. Los nombres funcionales ayudan a diseñar roles con el mínimo acceso necesario.

## Acceso común

| Permiso | Uso |
| --- | --- |
| `Common.App.Access` | Permite acceder a la aplicación. |
| `Common.App.RegularMode` | Habilita uso regular de la aplicación. |
| `Common.App.WorkMode` | Habilita modo trabajo cuando aplica. |

## Artículos

| Permiso | Uso |
| --- | --- |
| `Items.Products.Read` / `Items.Products.Write` | Leer o modificar productos. |
| `Items.Properties.Read` / `Items.Properties.Write` | Leer o modificar propiedades. |
| `Items.Skus.Read` / `Items.Skus.Write` | Leer o modificar artículos. |

## Almacén

| Permiso | Uso |
| --- | --- |
| `Warehouse.Locations.Read` / `Warehouse.Locations.Write` | Ubicaciones. |
| `Warehouse.Lots.Read` / `Warehouse.Lots.Write` | Lotes. |
| `Warehouse.Receipts.Read` / `Warehouse.Receipts.Write` | Recepciones. |
| `Warehouse.Shipments.Read` / `Warehouse.Shipments.Write` | Envíos. |
| `Warehouse.Counts.Read` / `Warehouse.Counts.Write` | Inventarios. |
| `Warehouse.Containers.Read` / `Warehouse.Containers.Write` | Contenedores. |

## Planificación

| Permiso | Uso |
| --- | --- |
| `Planning.Customers.Read` / `Planning.Customers.Write` | Clientes. |
| `Planning.Suppliers.Read` / `Planning.Suppliers.Write` | Proveedores. |
| `Planning.SalesOrders.Read` / `Planning.SalesOrders.Write` | Pedidos de venta. |
| `Planning.PurchaseOrders.Read` / `Planning.PurchaseOrders.Write` | Pedidos de compra. |
| `Planning.ProductionForecasts.Read` | Previsiones de fabricación. |
| `Planning.Skus.Read` / `Planning.Skus.Write` | Datos de planificación del artículo. |

## Producción

| Permiso | Uso |
| --- | --- |
| `Production.Recipes.Read` / `Production.Recipes.Write` | Recetas. |
| `Production.Recipes.Activate` | Activar versiones de receta. |
| `Production.Orders.Read` / `Production.Orders.Write` | Órdenes de fabricación. |
| `Production.Orders.Work` | Ejecutar operaciones. |
| `Production.Orders.Sequence` | Secuenciar o planificar órdenes. |
| `Production.Requests.Read` / `Production.Requests.Write` | Peticiones de fabricación. |
| `Production.Workstations.Read` / `Production.Workstations.Write` | Puestos de trabajo. |

## Personas y otros módulos

| Permiso | Uso |
| --- | --- |
| `People.Users.Read` | Consultar usuarios. |
| `People.Employees.Read` / `People.Employees.Write` | Empleados. |
| `People.ShiftScheduling.Read` / `People.ShiftScheduling.Write` | Turnos y calendarios. |
| `Files.Files.Upload` / `Files.Files.Read` / `Files.Files.Delete` | Archivos. |
| `Labels.Templates.Read` / `Labels.Templates.Write` | Plantillas de etiquetas. |
| `Agents.Questions.OtherUsers` | Permite aclaraciones de agentes para otros usuarios. |

<Warning>
  Da permisos de escritura solo cuando la persona debe cambiar datos. La lectura suele ser suficiente para supervisión, análisis o soporte.
</Warning>
