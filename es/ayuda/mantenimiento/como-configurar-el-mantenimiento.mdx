title: "Configurar el mantenimiento"
description: "Crea activos, acciones y causas para registrar trabajos de mantenimiento con criterios consistentes."
---

## Crear un activo

**Ruta:** **Mantenimiento \> Activos**

En la pantalla de activos puedes consultar los recursos introducidos en Fabricación y crear equipos o espacios físicos que necesiten mantenimiento.

<Note>
  **Ejemplos:** cortadoras, hornos industriales, líneas de ensamblaje o áreas de empaquetado.
</Note>

Para registrar un activo en el sistema completa los siguientes campos:

- **Tipo de activo:** selecciona la clase de elemento que vas a mantener.
- **Nombre:** asigna un nombre descriptivo que el equipo de planta pueda reconocer.
- **Ubicación o jerarquía:** indica si el activo está dentro de una zona, planta o almacén, o si forma parte de un equipo mayor.
- **Código de identificación:** introduce el código interno existente. Si no tienes uno, deja el campo en blanco para que el sistema lo genere.
- **Datos complementarios:** añade información como el fabricante, el modelo o el número de serie.

## Crear acciones

**Ruta:** **Mantenimiento \> Acciones \> Añadir nuevo**

Define las tareas reutilizables que pueden añadirse a un trabajo de mantenimiento.

<Note>
  **Ejemplos:** cambiar una cuchilla, limpiar filtros o sustituir correas.
</Note>

Indica el tiempo aproximado de ejecución y el tipo de resultado que debe registrar el operario.

## Crear causas

**Ruta:** **Mantenimiento \> Causas \> Añadir nuevo**

Define de forma estandarizada los motivos que hacen necesaria una intervención.

Esto simplifica el registro diario y permite identificar las averías más recurrentes.

<Note>
  **Ejemplos:** desgaste de correas, fallo eléctrico o suciedad.
</Note>

Cuando hayas creado la causa, asígnale las acciones habituales para resolver ese tipo de problema.

<Note>
  **Ejemplo:** si ante un desgaste de correas siempre realizas un cambio de correas, asocia la acción para que quede planificada en esa situación.
</Note>

<Note>
  Recuerda que en la columna de **Acciones** de todos estos campos puedes entrar a la línea creada para tener más información, modificar o añadir datos.
</Note>
