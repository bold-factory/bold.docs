title: "Editar un artículo"
description: "Actualiza un artículo sin romper stock, pedidos, producción ni trazabilidad histórica."
icon: "pencil"
---

Puedes editar un **Artículo** para corregir datos, completar propiedades o ajustar información de almacén y planificación. Algunas ediciones quedan limitadas cuando el artículo ya participa en operaciones trazables.

## Qué puedes cambiar normalmente

- Nombre y descripción.
- Imagen.
- Referencia externa.
- Valores de propiedades.
- Datos de proveedor y planificación.
- Configuración de almacén si no contradice movimientos existentes.

<Info>
  El código del artículo debe tratarse como identificador estable. Cambiarlo puede afectar búsquedas, integraciones y referencias operativas.
</Info>

## Editar datos básicos

<Steps>
  <Step title="Abre el artículo">
    Entra en **Catálogo > Artículos** y selecciona el artículo.
  </Step>
  <Step title="Revisa el estado">
    Si aparece como inválido, abre los errores para saber qué información falta.
  </Step>
  <Step title="Actualiza los campos">
    Cambia los datos necesarios. Si editas propiedades, asegúrate de respetar el tipo configurado.
  </Step>
  <Step title="Guarda y revisa impactos">
    Comprueba que el artículo sigue siendo válido y que planificación, stock o recetas muestran los datos esperados.
  </Step>
</Steps>

## Valores de propiedades

Los valores de propiedades diferencian artículos de un mismo producto. Por ejemplo, para la propiedad **Color**, un artículo puede tener **Rojo** y otro **Azul**.

El valor debe respetar la clase de la propiedad:

| Tipo de propiedad | Valor válido |
| --- | --- |
| Texto | Cualquier texto operativo. |
| Sí/no | Verdadero o falso. |
| Número | Cantidad numérica. |
| Fecha/hora | Fecha válida. |
| Selección de opción | Una opción existente de la propiedad. |

## Cambiar planificación

Los datos de planificación del artículo afectan a recomendaciones, compras y fabricación. Revisa el impacto antes de guardar cambios en artículos con demanda o suministro activo.

| Cambio | Impacto esperado |
| --- | --- |
| Origen **Compra** | Las faltas pueden aparecer en el Asistente de compras o generar pedidos directos. |
| Origen **Fabricación** | Las faltas pueden aparecer en el Asistente de fabricación o generar peticiones directas. |
| Origen **Ignorar** | Bold elimina previsiones de compra o fabricación para ese artículo. |
| Modo **Contra pedido** | Las demandas se cubren de forma vinculada cuando aplica. |
| Modo **Contra stock** | Las demandas se agrupan contra el stock disponible y proyectado. |
| Punto de pedido | Cambia cuándo se disparan recomendaciones. |
| Pedido mínimo y tamaño de lote | Cambian la cantidad propuesta. |
| Lead time | Cambia cuándo conviene lanzar la reposición. |

<Warning>
  Cambiar origen, modo o cantidades de planificación puede crear, actualizar o retirar previsiones. Revisa el stock proyectado después de guardar.
</Warning>

## Consolidar duplicados

Si Bold detecta dos artículos con el mismo producto y los mismos valores de propiedades, puede proponer consolidarlos. Consolidar deja un artículo como referencia válida y retira el duplicado, moviendo relaciones operativas al artículo que permanece.

<Warning>
  Consolida solo cuando ambos registros representan exactamente el mismo elemento físico. Si no son iguales, crea o completa una propiedad que explique la diferencia.
</Warning>
