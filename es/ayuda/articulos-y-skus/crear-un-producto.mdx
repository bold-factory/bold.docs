title: "Crear un producto"
description: "Crea una familia de artículos y define sus propiedades esperadas."
---

Un **Producto** agrupa artículos que pertenecen al mismo concepto comercial o técnico. Crea un producto cuando quieres organizar variantes bajo una misma familia.

## Cuándo crear un producto

Usa un producto cuando:

- vendes o fabricas variantes de una misma familia;
- quieres definir propiedades comunes para esos artículos;
- necesitas recetas de fabricación a nivel de familia;
- quieres analizar stock o actividad agrupada.

Si solo tienes un artículo aislado y no necesitas agrupar variantes, puedes trabajar con el artículo directamente.

## Crear el producto

<Steps>
  <Step title="Abre Catálogo">
    Entra en **Catálogo > Productos**.
  </Step>
  <Step title="Crea el registro">
    Haz clic en **Crear producto** y completa código, nombre, descripción y referencia externa si aplica.
  </Step>
  <Step title="Asigna propiedades">
    Añade propiedades como color, medida, material o acabado si el producto tiene variantes.
  </Step>
  <Step title="Crea artículos">
    Crea los artículos concretos de ese producto y completa los valores de cada propiedad.
  </Step>
</Steps>

## Campos habituales

| Campo | Uso |
| --- | --- |
| Código | Identificador de negocio único. |
| Nombre | Nombre visible para búsquedas y listados. |
| Descripción | Contexto adicional para administración interna. |
| Referencia externa | Id de otro sistema si existe integración. |
| Propiedades | Atributos que deben informar los artículos del producto. |

<Tip>
  Usa propiedades solo cuando cambien la definición física, comercial u operativa del artículo. No las uses como notas libres.
</Tip>

## Efecto de las propiedades

Cuando asignas una propiedad al producto, todos sus artículos deben tener un valor para esa propiedad. Si falta, el artículo queda inválido hasta completar la información.

No debería haber dos artículos activos con el mismo producto y los mismos valores de propiedades. Si representan exactamente el mismo elemento físico, consolídalos. Si no son iguales, falta una propiedad para explicar la diferencia.
