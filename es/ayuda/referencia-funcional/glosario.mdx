---
title: "Glosario"
description: "Definiciones rápidas de los conceptos principales de Bold."
---

Este glosario reúne términos que aparecen en los módulos de Bold. Úsalo para alinear vocabulario entre administración, almacén, planificación y producción.

| Término | Definición |
| --- | --- |
| Artículo | Unidad física que se compra, fabrica, almacena o vende. También puede aparecer como SKU en integraciones. |
| Producto | Familia o modelo que agrupa artículos relacionados. |
| Propiedad | Atributo que diferencia artículos, como color, formato o acabado. |
| Lote | Conjunto trazable de unidades de un artículo. |
| Ubicación | Lugar físico o lógico donde el stock entra, sale o se almacena. |
| Stock físico | Cantidad registrada actualmente en almacén. |
| Stock disponible | Stock que queda libre después de considerar compromisos. |
| Stock proyectado | Stock esperado al aplicar entradas y salidas futuras. |
| Demanda | Necesidad de stock generada por venta, consumo o previsión. |
| Suministro | Entrada esperada por compra, fabricación o previsión. |
| Pedido de compra | Compromiso con proveedor para recibir artículos. |
| Recepción | Documento que registra la entrada física de material. |
| Pedido de venta | Compromiso con cliente para entregar artículos. |
| Envío | Documento que registra la salida física de material. |
| Receta | Definición de cómo fabricar un artículo. |
| Versión de receta | Variante controlada de una receta. Solo una versión activa debe usarse como referencia principal. |
| Operación | Paso ejecutable dentro de una orden de fabricación. |
| Orden de fabricación | Trabajo real que transforma materiales en producto terminado o semielaborado. |
| Petición de fabricación | Necesidad de fabricar antes de lanzar una orden. |
| Puesto de trabajo | Recurso donde se ejecutan operaciones. |
| Co-producto | Salida adicional generada durante una fabricación. |
| Inventario | Recuento físico que regulariza diferencias de stock. |
| Rol | Conjunto de permisos asignable a usuarios. |
| Permiso | Capacidad concreta para leer, escribir o ejecutar acciones. |
| Clave API | Credencial técnica para integraciones externas. |
| Webhook | Notificación HTTP enviada cuando ocurre un evento. |

<Info>
  En la interfaz, Bold usa **Artículo** para el objeto operativo sobre el que se controla stock. En documentación técnica puede aparecer como `sku`.
</Info>
