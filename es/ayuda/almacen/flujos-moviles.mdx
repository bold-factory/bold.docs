Los **flujos móviles** permiten trabajar desde móvil o tablet cerca del material: consultar stock, contar, recibir, mover o preparar operaciones sin volver al ordenador.

Úsalos cuando el dato debe registrarse en el punto físico donde ocurre el movimiento.

## Antes de empezar

Prepara códigos de ubicación, etiquetas de artículo, lotes o bultos, conexión de red y permisos del usuario. El equipo debe poder identificar físicamente lo que escanea.

## Flujos habituales

| Flujo | Uso |
| --- | --- |
| Consultar stock | Ver cantidad por artículo, ubicación, lote o bulto. |
| Recibir | Registrar entrada cuando el material llega a almacén. |
| Mover | Cambiar ubicación, lote o bulto durante traslado interno. |
| Contar | Registrar recuentos de inventario desde el punto físico. |
| Preparar salida | Seleccionar stock para enviar o entregar. |

## Trabajar desde móvil

<Steps>
  <Step title="Abre el flujo correcto">
    Elige consulta, recepción, movimiento, inventario o salida según la operación real.
  </Step>
  <Step title="Escanea o busca el origen">
    Identifica ubicación, artículo, lote, bulto o documento relacionado.
  </Step>
  <Step title="Introduce cantidad">
    Registra la cantidad real. Revisa unidades y decimales antes de confirmar.
  </Step>
  <Step title="Escanea destino si aplica">
    Informa nueva ubicación o bulto cuando el material se traslada.
  </Step>
  <Step title="Confirma y revisa">
    Comprueba el resultado en stock o movimientos antes de cerrar el flujo.
  </Step>
</Steps>

## Qué cambia en Bold

Cada confirmación puede crear movimientos, actualizar ubicaciones, registrar conteos o cambiar el estado de documentos. Trabajar en móvil reduce transcripción, pero también exige validar el escaneo correcto antes de confirmar.

## Problemas habituales

- Si el código no se lee, revisa etiqueta, iluminación, formato y cámara.
- Si aparece la ubicación equivocada, comprueba que el código físico corresponde a la ubicación real.
- Si la cantidad no cuadra, revisa unidad, decimales y lote.
- Si falta permiso, revisa el rol del usuario para el flujo concreto.
