Almacén controla dónde está el stock, cómo entra, cómo sale y qué movimientos explican cada cambio. Es la base física para compras, ventas, planificación, fabricación y costes.

## Qué cambia el stock

| Acción | Efecto |
| --- | --- |
| Recibir material | Aumenta stock físico. |
| Enviar material | Reduce stock físico. |
| Mover stock | Cambia ubicación, lote o bulto sin cambiar el total. |
| Ajustar stock | Corrige una diferencia puntual. |
| Hacer inventario | Regulariza según recuento físico. |
| Consumir o producir | Actualiza stock desde fabricación. |

## Antes de empezar

Define ubicaciones y decide qué artículos trabajan con lotes. Si operas con móviles o tablets, prepara códigos de ubicación, bultos y flujos de lectura para evitar capturas manuales.
