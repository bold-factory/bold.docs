Un ajuste de stock corrige una diferencia puntual entre el stock registrado y la realidad física. Úsalo cuando sabes qué debe corregirse y no necesitas abrir un inventario completo.

Bold registra el ajuste como un inventario técnico de una sola línea. El historial conserva ubicación, lote, cantidad final indicada, diferencia aplicada y comentario.

## Cuándo usar un ajuste

Usa un ajuste para:

- corregir una diferencia pequeña detectada fuera de inventario;
- registrar una merma conocida;
- corregir una entrada o salida no registrada si no existe una reversión adecuada;
- ajustar una ubicación o lote tras una investigación.

<Warning>
  No uses ajustes para ocultar errores recurrentes. Si aparecen diferencias frecuentes, revisa recepción, picking, consumos, envíos o inventarios.
</Warning>

## Crear un ajuste

<Steps>
  <Step title="Identifica el stock afectado">
    Revisa artículo, ubicación y lote. Asegúrate de que la diferencia no se explica por un movimiento reciente.
  </Step>
  <Step title="Indica la cantidad final">
    Escribe la cantidad física que debe quedar en esa ubicación y lote. No escribas la diferencia.
  </Step>
  <Step title="Confirma el ajuste">
    Añade un comentario claro y guarda el ajuste para generar el movimiento trazable.
  </Step>
  <Step title="Revisa el resultado">
    Consulta stock y movimientos para confirmar que la cantidad final es correcta.
  </Step>
</Steps>

## Qué registra Bold

| Dato | Uso |
| --- | --- |
| Ubicación | Define dónde se corrige el saldo. |
| Lote | Mantiene trazabilidad cuando el artículo gestiona lotes. |
| Cantidad | Es el stock físico final que debe quedar. |
| Comentario | Explica el motivo de la corrección. |
| Ajuste aplicado | Diferencia entre el stock registrado y la cantidad final indicada. |

<Info>
  Si no seleccionas lote en un artículo sin gestión de lotes, Bold usa el lote interno desconocido. En artículos con lotes, selecciona el lote real para no perder trazabilidad.
</Info>

## Ajuste o inventario

| Situación | Usa |
| --- | --- |
| Una diferencia concreta en un artículo, ubicación y lote. | Ajuste de stock. |
| Muchas referencias en la misma zona. | Inventario. |
| Recuento periódico o cierre de periodo. | Inventario. |
| Corrección tras una investigación con causa clara. | Ajuste de stock. |

## Buenas prácticas

| Práctica | Motivo |
| --- | --- |
| Añade motivo y notas | Facilita auditoría y análisis posterior. |
| Ajusta el lote correcto | Evita perder trazabilidad por origen. |
| Ajusta en la ubicación correcta | Evita descuadres entre zonas. |
| Revisa permisos | Limita ajustes a perfiles responsables. |

<Tip>
  Si la diferencia afecta a muchos artículos o ubicaciones, crea un inventario. Es más claro que una cadena de ajustes sueltos.
</Tip>
