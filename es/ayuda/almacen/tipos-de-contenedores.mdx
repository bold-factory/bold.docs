Los **tipos de contenedores** describen formatos físicos que usas para mover o almacenar material: palets, cajas, bidones, bolsas, carros, jaulas u otros.

Después, cada **bulto** concreto se crea a partir de un tipo de contenedor y se ubica en almacén.

## Antes de empezar

Define tipos solo para formatos que quieras identificar, imprimir, mover o analizar. No hace falta crear un tipo para cada proveedor si el formato físico es el mismo.

## Campos habituales

| Campo | Uso |
| --- | --- |
| Nombre | Identifica el formato, como **Palet europeo** o **Caja pequeña**. |
| Descripción | Aclara cuándo se usa. |
| Anchura, longitud y altura | Ayudan a diferenciar formatos y preparar diseños o etiquetas. |

## Crear tipos de contenedores

<Steps>
  <Step title="Lista formatos reales">
    Revisa qué cajas, palets, carros o contenedores usa el almacén.
  </Step>
  <Step title="Crea cada tipo">
    Usa nombres claros que el equipo reconozca durante recepción, movimiento o preparación.
  </Step>
  <Step title="Añade dimensiones">
    Completa medidas cuando ayuden a impresión, capacidad o identificación física.
  </Step>
  <Step title="Crea bultos desde el tipo">
    Usa el tipo como plantilla para los bultos concretos que se moverán.
  </Step>
  <Step title="Revisa etiquetas">
    Si imprimes etiquetas, comprueba que el tipo tenga los datos necesarios.
  </Step>
</Steps>

## Qué cambia en Bold

El tipo de contenedor no mueve stock por sí mismo. Sirve para clasificar bultos y mantener criterios homogéneos al identificar material físico.

## Problemas habituales

- Si hay muchos tipos parecidos, agrúpalos por formato físico real.
- Si una etiqueta no muestra datos esperados, revisa el tipo y la plantilla de diseño.
- Si un bulto se creó con el tipo incorrecto, corrígelo antes de usarlo en movimientos trazables.
- Si necesitas capacidad o dimensiones exactas, define una convención de medida común.
