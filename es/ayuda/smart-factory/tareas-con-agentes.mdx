title: "Revisar tareas con agentes"
description: "Revisa trabajos ejecutados por agentes, continúa conversaciones y audita créditos."
---

Una **tarea con agente** es un trabajo delegado a un agente especializado. Puede iniciarse manualmente, desde una automatización o mediante un evento. La tarea conserva instrucciones, mensajes, archivos adjuntos, llamadas a herramientas, aclaraciones, resultado y créditos consumidos.

## Qué muestra el historial

| Campo | Uso |
| --- | --- |
| Estado | Indica si la tarea está pendiente, en ejecución, completada, requiere acción o falló. |
| Agente | Agente que ejecuta el trabajo. |
| Inicio | Fecha y hora de creación. |
| Última interacción | Último cambio relevante de la tarea. |
| Créditos | Consumo asociado a la ejecución. |
| Caducada | Las tareas caducan tras 28 días de inactividad. |

## Estados

| Estado | Qué hacer |
| --- | --- |
| Pendiente | Espera a que el agente empiece a procesar. |
| En ejecución | No respondas todavía. El agente está trabajando. |
| Requiere acción | Responde a la aclaración o decisión que pide el agente. |
| Completada | Revisa el resultado y continúa solo si necesitas ampliar el trabajo. |
| Fallida | Revisa el error, ajusta instrucciones o crea una tarea nueva. |

## Crear una tarea

<Steps>
  <Step title="Abre Tareas con agentes">
    Entra en **Automatizaciones** y abre **Tareas con agentes**.
  </Step>
  <Step title="Crea una tarea nueva">
    Selecciona **Nueva tarea**.
  </Step>
  <Step title="Elige el agente">
    Selecciona un agente con permisos para los módulos que necesita la tarea. Si el agente no tiene permisos, probablemente no podrá completarla.
  </Step>
  <Step title="Añade instrucciones">
    Escribe el objetivo, límites, formato esperado y datos de referencia. Las instrucciones adicionales admiten hasta `3000` caracteres.
  </Step>
  <Step title="Adjunta archivos si hacen falta">
    Añade documentos, imágenes o archivos que el agente deba usar como contexto.
  </Step>
  <Step title="Revisa el resultado">
    Abre la tarea para ver mensajes, herramientas usadas, aclaraciones y resultado final.
  </Step>
</Steps>

## Buenas instrucciones

| Caso | Instrucción útil |
| --- | --- |
| Resumen operativo | `Resume órdenes de fabricación en progreso con retrasos, causas probables y próximos pasos.` |
| Auditoría de mantenimiento | `Revisa partes abiertos del activo ENV-01 y destaca los que requieren acción del responsable.` |
| Preparación de informe | `Con estos datos adjuntos, identifica inconsistencias antes de importarlos.` |

## Automatizaciones y eventos

Las automatizaciones ejecutan agentes sin crear la tarea manualmente. Cada ejecución crea una tarea con historial propio, estado, resultado y créditos consumidos.

| Tipo | Cuándo usarla | Qué revisar |
| --- | --- | --- |
| Periódica | Para revisiones o tareas recurrentes con frecuencia diaria, semanal, mensual, trimestral, semestral o anual. | Próxima ejecución, última ejecución, responsable, zona horaria y número de ejecuciones. |
| Por evento | Para reaccionar a cambios publicados por Bold, como pedidos, órdenes, stock o partes. | Evento, versión, condición, agente responsable y si está habilitada. |

Las automatizaciones periódicas se ejecutan cuando llega la próxima fecha programada. Las automatizaciones por evento se ejecutan cuando el evento coincide y la condición configurada se cumple.

Para tareas críticas, revisa las primeras ejecuciones antes de confiar en la automatización. Ajusta instrucciones, permisos y nivel de inteligencia si el agente omite pasos o usa demasiado contexto.

<Warning>
  Usa agentes personalizados para automatizaciones. Los agentes de sistema no pueden programarse ni suscribirse a eventos.
</Warning>

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| La tarea requiere acción | Abre la tarea y responde a las preguntas pendientes. |
| No puedes continuar | Puede estar en ejecución, fallida, esperando aclaración específica o caducada. |
| El agente no encuentra datos | Revisa permisos del agente y si los datos existen en Bold. |
| Consume demasiados créditos | Baja el nivel de inteligencia o acota mejor las instrucciones. |
| La tarea caducó | Crea una nueva tarea con el contexto necesario. |
