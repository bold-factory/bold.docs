---
title: "Consultar informes Power BI"
description: "Consulta y adapta informes Power BI sobre datos de Bold."
---

Los **informes Power BI** usan Power BI integrado para analizar datos de Bold con visualizaciones, filtros y campos preparados. Úsalos para análisis periódico, seguimiento de indicadores y vistas de dirección.

<Info>
  Los informes Power BI no son una vista en tiempo real. Se refrescan según la programación del entorno, normalmente de forma periódica durante el horario laboral.
</Info>

## Qué puedes hacer

| Acción | Uso |
| --- | --- |
| Abrir un informe | Consulta visualizaciones y filtros existentes. |
| Crear informe | Parte del modelo de datos predefinido de Bold. |
| Duplicar informe | Crea una copia para modificarla sin alterar el original. |
| Editar y guardar | Ajusta campos, visualizaciones y filtros cuando tengas permisos. |
| Exportar datos | Extrae datos según las capacidades habilitadas en el informe. |
| Eliminar | Borra informes editables. Esta acción no se puede deshacer. |

## Cuándo usar informes y cuándo usar cuadros de mando

| Necesidad | Usa |
| --- | --- |
| Analizar tendencias, comparar periodos o preparar informes | **Informes Power BI**. |
| Ver estado vivo de planta, operaciones en curso u OEE operativo | [Cuadros de mando de fabricación](/es/conceptos/fabricacion/cuadros-de-mando). |
| Investigar un registro concreto | El módulo operativo correspondiente, por ejemplo órdenes, stock o partes. |

## Crear o adaptar un informe

<Steps>
  <Step title="Abre Informes">
    Entra en **Smart Factory** y abre **Informes**.
  </Step>
  <Step title="Elige el punto de partida">
    Abre un informe existente, crea uno nuevo o duplica uno parecido. El nombre `Base` está reservado.
  </Step>
  <Step title="Edita visualizaciones">
    En modo edición, selecciona campos del modelo, añade visualizaciones y configura filtros.
  </Step>
  <Step title="Guarda cambios">
    Guarda cuando el informe represente la vista que necesita tu equipo.
  </Step>
  <Step title="Valida con datos reales">
    Compara el informe con registros de Bold para confirmar que filtros, fechas y unidades son correctos.
  </Step>
</Steps>

## Permisos e integraciones

| Acción | Permiso necesario |
| --- | --- |
| Abrir informes | Acceso de lectura a informes. |
| Crear, duplicar o eliminar informes | Acceso de edición a informes. |

## Buenas prácticas

- Duplica informes antes de hacer cambios relevantes.
- Usa nombres orientados al uso, por ejemplo `Coste por activo` o `Producción semanal`.
- Incluye filtros visibles para fechas, área, artículo, activo o estado cuando el informe se use por varios equipos.
- No uses informes Power BI para decisiones que requieren estado en vivo de planta.

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| El dato parece desactualizado | Revisa la última actualización del modelo y usa cuadros de mando si necesitas tiempo real. |
| No aparece un informe | Puede no haber informes disponibles o faltar permisos de Analytics. |
| No puedes eliminarlo | Solo los informes editables pueden eliminarse. |
| Un filtro cambia demasiado el resultado | Revisa fechas, estado de registros, unidades y campos usados en la visualización. |
