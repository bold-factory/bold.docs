title: "Conversar con Boldy"
description: "Conversa con el asistente para consultar información de Bold sin navegar por menús."
---

**Boldy** es el asistente conversacional de Bold. Úsalo para consultar información en lenguaje natural, resumir datos y orientarte sin recorrer todos los menús.

## Cuándo usar Boldy

| Necesidad | Ejemplo |
| --- | --- |
| Localizar datos | `¿Qué pedidos de venta están pendientes esta semana?` |
| Resumir información | `Resume las órdenes de fabricación con retraso.` |
| Entender un proceso | `Explícame qué significa stock proyectado.` |
| Preparar una búsqueda | `Ayúdame a encontrar partes de mantenimiento de la envasadora.` |

Boldy responde con el contexto y permisos disponibles para tu usuario. Si no tiene datos suficientes, debe pedir aclaración o indicar qué falta.

## Iniciar una conversación

<Steps>
  <Step title="Abre Boldy">
    Entra en **Automatizaciones** y abre **Boldy**.
  </Step>
  <Step title="Revisa el historial">
    Abre una conversación anterior si quieres continuar con el mismo contexto.
  </Step>
  <Step title="Crea una conversación nueva">
    Usa **Nueva** cuando cambie el objetivo o no quieras arrastrar contexto anterior.
  </Step>
  <Step title="Formula la petición">
    Indica módulo, periodo, estado o identificador cuando lo conozcas.
  </Step>
  <Step title="Verifica el resultado">
    Para decisiones operativas, abre los registros citados y confirma estados, fechas y cantidades.
  </Step>
</Steps>

## Cómo preguntar mejor

| Menos útil | Más útil |
| --- | --- |
| `¿Cómo va producción?` | `Resume las OF en progreso de esta semana y destaca retrasos.` |
| `Mira stock` | `Qué SKUs están por debajo del punto de reorden en el almacén principal.` |
| `Dime mantenimiento` | `Lista partes abiertos del activo ENV-01 con causa y última interacción.` |

## Diferencia con tareas con agentes

| Herramienta | Uso principal |
| --- | --- |
| Boldy | Conversación directa y rápida con el asistente personal. |
| Tareas con agentes | Trabajos delegados a agentes especializados, con estado, archivos, herramientas, aclaraciones y créditos. |

<Tip>
  Usa Boldy para explorar o preguntar. Usa una tarea con agente cuando el trabajo sea repetible, largo, necesite archivos o deba quedar auditado como ejecución delegada.
</Tip>

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| La respuesta es demasiado general | Añade módulo, fechas, estado, código, SKU, activo o ubicación. |
| Boldy no encuentra un dato | Revisa permisos y si el dato existe en el módulo operativo. |
| Mezcla contextos de conversaciones anteriores | Crea una conversación nueva. |
| Necesitas que ejecute una tarea larga | Usa [Consultar o delegar con IA](/es/ayuda/smart-factory/consultar-o-delegar-con-ia). |
