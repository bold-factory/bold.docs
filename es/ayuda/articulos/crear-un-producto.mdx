---
title: "Crear un producto"
description: "Crea una familia de artículos y define sus propiedades esperadas."
---

Un **Producto** agrupa artículos que pertenecen al mismo concepto comercial o técnico. Crea un producto cuando quieres organizar variantes bajo una misma familia.

## Cuándo crearlo

Usa un producto cuando:

- vendes o fabricas variantes de una misma familia;
- quieres definir propiedades comunes para esos artículos;
- necesitas recetas de fabricación a nivel de familia;
- quieres analizar stock o actividad agrupada.

> Si solo tienes un artículo aislado y no necesitas agrupar variantes, puedes trabajar con el artículo directamente.

## Pasos a seguir

**Catálogo \> Productos \> Crear Nuevo**

### 1. Introducir datos

**Nombre:** es el identificador principal y más visible de lo que fabricas o almacenas.

**Código** (opcional): es el identificador alfanumérico. Evita confusiones si tienes dos productos con nombres similares (por ejemplo, si tienes el mismo tornillo pero de dos materiales distintos). Además, sirve para que en el futuro puedas usar lectores de códigos de barras o enlazar este software con sistemas de facturación o ERPs.

**Descripción** (opcional): es un espacio libre de texto para añadir detalles específicos del producto que no caben en el nombre. Puedes poner medidas, materiales, colores o incluso advertencias de almacenamiento.

### 2. Asignar propiedades

Dentro del producto creado puedes seleccionar todas las propiedades que definiste anteriormente. Esa elección depende de si la planta necesita ese dato técnico para poder ejecutar el trabajo correctamente.

> **_Ejemplo:_** Si fabricas camisetas, el "Color" y la "Talla" son propiedades obligatorias porque necesitas saber si estás fabricando una camiseta Roja M o una Azul XL.

## Efecto de las propiedades

Cuando asignas una propiedad al producto, todos sus artículos deben tener un valor para esa propiedad. Si falta, el artículo queda inválido hasta completar la información.

No debería haber dos artículos activos con el mismo producto y los mismos valores de propiedades. Si representan exactamente el mismo elemento físico, consolídalos. Si no son iguales, falta una propiedad para explicar la diferencia.
