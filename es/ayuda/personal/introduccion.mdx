El módulo Personal mantiene empleados y programación básica para fabricación, mantenimiento y análisis de productividad.

## Flujo recomendado

<Steps>
  <Step title="Configura la base">
    Crea los datos maestros que el módulo necesita para operar.
  </Step>
  <Step title="Registra actividad">
    Usa los registros funcionales para conservar histórico y alimentar informes.
  </Step>
  <Step title="Revisa resultados">
    Consulta estados, costes, carga, permisos o automatizaciones para detectar bloqueos.
  </Step>
</Steps>

## Antes de empezar

Define responsables, permisos y datos mínimos. Evita crear duplicados si el dato ya existe en otro módulo.
