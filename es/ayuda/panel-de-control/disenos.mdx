Usa **Diseños** para crear plantillas imprimibles que identifican objetos de Bold con texto, códigos QR, códigos de barras, imágenes y datos dinámicos.

## Tipos de plantilla

| Tipo | Uso habitual |
| --- | --- |
| Artículo | Etiquetas de SKU o producto almacenable. |
| Lote | Trazabilidad de lote en almacén o producción. |
| Bulto | Identificación de contenedores o embalajes. |
| Línea de recepción | Etiquetas para material recibido. |
| Operación | Identificación de operaciones o instrucciones de planta. |
| Operaciones de orden | Etiquetas agrupadas para una orden. |
| Empleado | Identificación de personal operativo. |
| Estación de trabajo | Identificación de recursos de fabricación. |
| Grupo de operaciones | Identificación de agrupaciones de operaciones. |

## Elementos disponibles

| Elemento | Uso |
| --- | --- |
| Forma | Separadores o fondos simples. |
| Texto fijo | Texto escrito en la plantilla. |
| Texto variable | Campo dinámico de Bold o metadato. |
| Código QR | Uno o varios valores codificados. |
| Código de barras | Campo dinámico legible por escáner. |
| Imagen | Imagen vinculada cuando el tipo de plantilla lo permite. |

## Crear una plantilla

<Steps>
  <Step title="Abre Diseños">
    Entra en **Panel de control** y abre **Diseños**.
  </Step>
  <Step title="Crea la plantilla">
    Define nombre, tipo de plantilla, ancho y alto del lienzo en milímetros.
  </Step>
  <Step title="Añade elementos">
    Inserta texto, QR, código de barras, forma, imagen o valores dinámicos.
  </Step>
  <Step title="Ajusta posición y tamaño">
    Arrastra elementos dentro del lienzo y valida que no se salgan del área imprimible.
  </Step>
  <Step title="Descarga una prueba">
    Genera una muestra y comprueba legibilidad en el tamaño real.
  </Step>
  <Step title="Usa la plantilla en operación">
    Imprime desde el módulo que corresponda: artículo, lote, bulto, estación u orden.
  </Step>
</Steps>

## Buenas prácticas

- Diseña para el tamaño real de impresión, no para la vista ampliada en pantalla.
- Prioriza el dato que el operario o lector necesita identificar.
- Prueba QR y códigos de barras con el lector usado en planta o almacén.
- Usa metadatos solo cuando el equipo sepa mantenerlos.
- Evita plantillas con demasiada información.

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| El código no se lee | Aumenta tamaño, contraste o margen alrededor del QR o código de barras. |
| Falta un valor dinámico | Revisa el tipo de plantilla y la clave seleccionada. |
| El diseño se corta al imprimir | Revisa ancho, alto y márgenes físicos de la impresora. |
| No puedes guardar | Revisa permisos de etiquetas y que la plantilla tenga nombre y tamaño válidos. |
