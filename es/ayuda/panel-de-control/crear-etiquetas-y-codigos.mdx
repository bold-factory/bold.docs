---
title: "Crear etiquetas y códigos"
description: "Prepara etiquetas imprimibles con códigos para artículos, lotes, ubicaciones, bultos o recursos."
---

Crea etiquetas y códigos cuando el equipo vaya a identificar o escanear elementos físicos. Una buena etiqueta reduce errores en recepción, movimientos, fabricación y envíos.

## Decidir qué etiquetar

<Steps>
  <Step title="Elige el objeto">
    Decide si la etiqueta representa artículo, lote, ubicación, bulto, contenedor, estación, herramienta u orden.
  </Step>
  <Step title="Define el código">
    Usa una referencia estable y única para escaneo. Evita códigos que cambien con frecuencia.
  </Step>
  <Step title="Añade información humana">
    Incluye nombre corto, descripción, lote, ubicación o fecha solo si ayuda al trabajo físico.
  </Step>
  <Step title="Diseña el tamaño">
    Ajusta formato a impresora, superficie y distancia de lectura.
  </Step>
  <Step title="Prueba con escáner real">
    Imprime una muestra y valida lectura en almacén o planta.
  </Step>
</Steps>

## Usos habituales

| Etiqueta | Uso |
| --- | --- |
| Artículo | Identificar material en recepción o preparación. |
| Lote | Mantener trazabilidad en consumo, envío o inventario. |
| Ubicación | Registrar movimientos con menos errores. |
| Bulto o contenedor | Mover una unidad física agrupada. |
| Estación o recurso | Abrir operaciones desde modo de trabajo. |
| Orden u operación | Localizar trabajo mediante escaneo. |

<Tip>
  Prioriza etiquetas que eviten errores repetidos. No etiquetes todo si el equipo no va a escanearlo o leerlo en el proceso.
</Tip>
