Los **Agentes** son asistentes de IA configurables para ejecutar tareas sobre Bold con instrucciones y permisos concretos. Úsalos para trabajos repetibles, análisis con varios pasos o automatizaciones que deben quedar auditadas.

## Qué puedes configurar

| Campo | Uso |
| --- | --- |
| Nombre | Identifica el agente para usuarios y tareas. |
| Nivel de inteligencia | Ajusta capacidad de razonamiento y consumo de créditos. |
| Instrucciones | Define objetivo, límites y forma de responder. |
| Permitir preguntas | Permite que el agente pida aclaraciones cuando falte información. |
| Memoria | Guarda contexto persistente del agente cuando está habilitada. |
| Permisos | Determinan qué herramientas y datos puede usar. |
| Automatizaciones | Ejecutan el agente de forma programada. |

<Warning>
  Un agente solo debe tener los permisos que necesita. Si le das permisos amplios, podrá acceder a más datos y acciones durante sus tareas.
</Warning>

## Crear un agente

<Steps>
  <Step title="Abre Agentes">
    Entra en **Panel de control** y abre **Agentes**.
  </Step>
  <Step title="Crea el agente">
    Define nombre, nivel de inteligencia, instrucciones, preguntas y memoria.
  </Step>
  <Step title="Configura permisos">
    Marca solo los módulos y acciones necesarios. Puedes usar sugerencias de permisos si las instrucciones están claras.
  </Step>
  <Step title="Prueba con una tarea manual">
    Crea una [tarea con agente](/ayuda/smart-factory/tareas-con-agentes) y revisa si usa los datos correctos.
  </Step>
  <Step title="Ajusta instrucciones">
    Corrige instrucciones si el agente pide demasiadas aclaraciones, omite pasos o entrega un formato poco útil.
  </Step>
</Steps>

## Elegir nivel de inteligencia

| Nivel | Cuándo usarlo |
| --- | --- |
| Bajo | Consultas directas, resúmenes simples o tareas muy acotadas. |
| Medio | Tareas operativas habituales con equilibrio entre coste y razonamiento. |
| Alto | Análisis con varios pasos, instrucciones ambiguas o tareas que combinan varias áreas. |

Empieza por el nivel más bajo que cumpla la tarea. Sube el nivel si el agente se queda corto, omite pasos o necesita razonar con más contexto.

## Automatizaciones

Las automatizaciones ejecutan agentes de forma recurrente. Puedes definir frecuencia diaria, semanal, mensual, trimestral, semestral o anual, con horas y reglas de calendario.

Cada ejecución crea una tarea con agente. La tabla muestra agente, estado, próxima ejecución, última ejecución, ejecuciones acumuladas y responsable.

<Steps>
  <Step title="Crea la automatización">
    Selecciona agente, responsable, frecuencia y horas de ejecución.
  </Step>
  <Step title="Define reglas de calendario">
    Para programaciones semanales, elige días. Para mensuales, elige día del mes, último día o patrón como primer lunes.
  </Step>
  <Step title="Revisa permisos">
    Si el agente no tiene permisos, probablemente no podrá completar la automatización.
  </Step>
  <Step title="Activa y observa">
    Revisa las primeras tareas generadas antes de confiar en el proceso.
  </Step>
</Steps>

## Agentes de sistema

La lista puede incluir agentes de **Sistema**. Estos agentes son internos y no se eliminan desde la configuración. Los agentes creados por usuarios se pueden editar o retirar según permisos.

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| El agente no puede completar una tarea | Revisa permisos, instrucciones y datos disponibles. |
| Consume demasiados créditos | Baja el nivel de inteligencia o acota la tarea. |
| Pide aclaraciones continuamente | Mejora instrucciones o desactiva preguntas si el flujo debe fallar antes de continuar. |
| La automatización no se ejecuta | Revisa si está habilitada y si Bold puede calcular la próxima ejecución. |
| No puedes eliminar un agente | Puede ser un agente de sistema o faltarte permiso. |
