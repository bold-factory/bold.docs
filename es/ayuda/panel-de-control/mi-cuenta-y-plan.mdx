Usa **Mi cuenta y plan** para revisar la suscripción activa, los límites de uso y el consumo mensual visible para administradores.

## Qué puedes revisar

| Bloque | Qué muestra |
| --- | --- |
| Plan de suscripción | Nombre del plan activo. |
| Límites permanentes | Empleados, SKUs, ubicaciones y capacidades incluidas. |
| Uso mensual | Créditos de IA usados en el periodo seleccionado. |
| Facturación | Acceso al portal de facturación cuando está disponible. |

<Info>
  La pantalla de plan solo está disponible para administradores. Si no tienes acceso, pide a un administrador que revise el plan o tus permisos.
</Info>

## Revisar límites y uso

<Steps>
  <Step title="Abre Panel de control">
    Entra en **Panel de control** y abre **Mi cuenta y plan**.
  </Step>
  <Step title="Revisa el plan activo">
    Comprueba el nombre del plan y si las capacidades que necesitas están incluidas.
  </Step>
  <Step title="Comprueba límites permanentes">
    Revisa empleados, SKUs y ubicaciones actuales frente al límite contratado.
  </Step>
  <Step title="Consulta créditos de IA">
    Cambia de mes para ver el consumo mensual. El contador se reinicia al inicio de cada mes.
  </Step>
  <Step title="Gestiona facturación">
    Usa **Gestionar facturación** si necesitas abrir el portal de facturación.
  </Step>
</Steps>

## Límites que afectan al trabajo diario

| Límite o capacidad | Impacto |
| --- | --- |
| Empleados | Afecta a personal, asignaciones, tiempos y costes de mano de obra. |
| SKUs | Afecta al catálogo, stock, planificación, fabricación y compras. |
| Ubicaciones | Afecta a almacén, recepciones, movimientos e inventarios. |
| Planes de producción | Habilita planificación avanzada de producción cuando el plan lo incluye. |
| Gestión de múltiples lotes | Habilita trazabilidad por varios lotes cuando el plan lo incluye. |
| Créditos IA | Afecta a Boldy, tareas con agentes y automatizaciones. |

## Problemas habituales

| Problema | Qué revisar |
| --- | --- |
| No ves la página | Debes ser administrador. |
| Un límite aparece cerca del máximo | Revisa si puedes archivar datos inactivos o si necesitas ajustar el plan. |
| Los créditos de IA suben rápido | Revisa agentes, tareas automáticas y nivel de inteligencia. |
| No hay plan configurado | Revisa facturación o contacta con el responsable de la cuenta. |
