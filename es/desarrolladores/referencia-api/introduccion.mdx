La referencia de API de Bold se publica desde el backend en `https://api.bold-factory.com/docs`. Úsala para consultar endpoints, parámetros, esquemas, ejemplos y pruebas interactivas.

<Card title="Abrir documentación de API" icon="external-link" href="https://api.bold-factory.com/docs">
  Accede a la documentación publicada por el backend.
</Card>

## Qué contiene

- Endpoints de la versión pública.
- Parámetros de ruta, query y body.
- Esquemas de request y response.
- Modelos y códigos de respuesta.
- Pruebas interactivas contra el entorno de API.

## Antes de llamar a la API

Necesitas una clave API o un token bearer válido. Para integraciones servidor a servidor, crea una clave desde **Panel de control > Acceso > Claves API** y envíala en la cabecera `X-Api-Key`.
