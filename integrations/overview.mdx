No single system can cover every need of a manufacturing business. Bold Factory is designed to work alongside the tools you already use — your ERP, your customer portal, your purchasing system — so data flows freely between them instead of living in silos. Whether you need a simple scheduled sync or a fully real-time bidirectional integration, Bold gives you the building blocks to make it happen.

## Integration options

Bold provides three complementary integration paths. You can use them independently or combine them to build exactly the data flows your operation requires.

<CardGroup cols={3}>
  <Card title="Public REST API" icon="code" href="/integrations/public-api">
    Pull data into Bold or push data out to any system with a documented REST API and Swagger reference.
  </Card>
  <Card title="Webhooks" icon="bolt" href="/integrations/webhooks">
    Receive instant HTTP notifications when events happen in Bold — no polling, no overnight batch jobs.
  </Card>
  <Card title="ERP Connectivity" icon="building" href="/integrations/erp-connectivity">
    Bidirectional sync with SAP, Sage, Navision, A3, and other major ERP platforms.
  </Card>
</CardGroup>

## Supported ERPs

Bold ships with proven connectors for the ERP platforms most commonly used by manufacturing SMEs. Out-of-the-box support includes:

- **SAP** (S/4HANA and Business One)
- **Sage** (50, 100, 200, and X3)
- **Microsoft Dynamics NAV / Business Central** (Navision)
- **A3**
- Other platforms via the public API

<Tip>
  Don't see your ERP listed? Bold's public API is platform-agnostic. If your ERP exposes a REST or SOAP interface, an integration is almost certainly possible. [Get in touch](https://bold-factory.com/en/contact) and the Bold team will advise you.
</Tip>

## Choose how you implement

You have two paths to a live integration. Pick the one that fits your team and your timeline.

<CardGroup cols={2}>

## Where to start

<Steps>
  <Step title="Identify your integration points">
    List the systems Bold needs to exchange data with and decide which data flows in which direction.
  </Step>
  <Step title="Pick your integration method">
    Use the REST API for request-driven sync, webhooks for event-driven real-time notifications, or a pre-built ERP connector for common platforms.
  </Step>
  <Step title="Get your API key">
    Log in to Bold Factory, navigate to **Settings → API**, and generate your API key. Your Account Manager can also provide one.
  </Step>
  <Step title="Build or request">
    Implement the integration yourself using the docs below, or [contact Bold](https://bold-factory.com/en/contact) to request a managed quote.
  </Step>
</Steps>
